Teams Transcript Bookmarklet

1. Install (drag to bookmarks bar)

Teams Transcript

Or manually create a bookmark and paste the URL below into the address field:

javascript:(function(){var old=document.getElementById('__transcript_script');if(old)old.remove();var local='http://localhost:9985/script.js?_='+Date.now();var remote='https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/script.js?_='+Date.now();function inject(src){var s=document.createElement('script');s.id='__transcript_script';s.src=src;document.head.appendChild(s);}fetch(local,{mode:'no-cors'}).then(function(){inject(local);}).catch(function(){inject(remote);});})()

2. Start local server

cd /Users/btsai/git/teams_transcript_bookmarklet
python3 -m http.server 9985

3. Usage

Join a Teams meeting with transcription on, then click the bookmarklet. Check the DevTools console for [Transcript] Observer started.

To export the transcript, run in the console:

copy(JSON.stringify(window.transcript, null, 2))

To stop the observer:

window.myObserver.disconnect()

4. Update GitHub URL

Once the repo is public, replace YOUR_USERNAME/YOUR_REPO in the bookmarklet href above with your actual GitHub username and repo name.
